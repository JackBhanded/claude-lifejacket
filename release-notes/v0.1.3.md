# Claude Lifejacket v0.1.3 — crash fix

> Paste this into the GitHub release body for the `v0.1.3` tag.

A quick fix on top of v0.1.2's new look.

### Fixed

- **The app window no longer crashes on launch.** The v0.1.2 redesign left a reference to a removed colour constant in the window's refresh path, so opening the app threw a `NameError` on Windows. It's restored — the window opens cleanly, with the same new glass look and dark mode.

### Get it

Download **`Claude Lifejacket.exe`** below and double-click — no Python, no terminal. (Windows SmartScreen warns on the unsigned exe → **More info → Run anyway**.)
